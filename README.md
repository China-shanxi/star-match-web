# 星座运势&姓名配对测算网站
纯前端静态网页，GitHub Pages一键部署
## 新增全部功能
1. 星空粒子闪烁动态背景动画
2. 十二星座高清图标自动展示
3. 生日输入自动匹配对应星座
4. 免费公开API实时获取每日真实星座运势（不再静态文字）
5. LocalStorage本地存储测算历史记录，最多保存20条
6. 一键复制文本分享运势/配对结果
7. 适配手机/电脑移动端

## 使用方法
1. Clone仓库到本地
2. 直接打开 index.html 本地预览
3. 仓库开启 GitHub Pages 在线访问

## 技术栈
HTML5 + CSS3 + 原生JavaScript
存储：LocalStorage
接口：免费公开星座运势API
动画：CSS粒子星空闪烁
